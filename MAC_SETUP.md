# TeensyTalkV2 — Mac Setup: Generating MBROLA Phoneme WAV Files

This guide walks you through generating the 34 phoneme WAV files needed by TeensyTalkV2
on a Mac. These WAV files go on the Teensy SD card and are played back in sequence to
produce speech.

---

## Overview

TeensyTalkV2 uses the **MBROLA us2** voice database (American English male, 16000 Hz).
Instead of running MBROLA on the Teensy, you generate the individual phoneme WAV files
once on your Mac, then copy them to the SD card. The Teensy firmware simply plays them
back in the right order.

**Pipeline:** eSpeak-ng converts text → phoneme tokens → MBROLA renders each phoneme → WAV file

---

## Prerequisites

You need:
- A Mac with [Homebrew](https://brew.sh) installed
- Python 3 (comes with macOS or via `brew install python`)
- About 5 MB free on your SD card

---

## Step 1 — Install eSpeak-ng

```bash
brew install espeak-ng
```

Verify it works:

```bash
espeak-ng --version
```

---

## Step 2 — Install MBROLA

MBROLA is not in Homebrew, so install it manually:

```bash
# Download the macOS binary
curl -L https://github.com/numediart/MBROLA/releases/download/3.3/mbrola-mac -o /usr/local/bin/mbrola
chmod +x /usr/local/bin/mbrola
```

Verify it works:

```bash
mbrola -h
```

> **Note:** On Apple Silicon (M1/M2/M3) Macs, you may need Rosetta or to build from
> source. If the binary above fails, build from source:
> ```bash
> git clone https://github.com/numediart/MBROLA.git
> cd MBROLA
> make
> sudo cp Bin/mbrola /usr/local/bin/
> ```

---

## Step 3 — Download the MBROLA us2 Voice Database

```bash
# Create a working directory
mkdir -p ~/mbrola_work
cd ~/mbrola_work

# Download the us2 voice database
curl -L https://github.com/numediart/MBROLA-voices/raw/master/data/us2/us2 -o us2
```

The `us2` file is a binary diphone database (~3.5 MB). Keep it in `~/mbrola_work/`.

---

## Step 4 — Understand the MBROLA .pho Format

MBROLA takes a plain-text `.pho` file as input. Each line has:

```
PHONEME  DURATION_MS  [PITCH_PERCENT PITCH_HZ ...]
```

For a single isolated phoneme WAV, use a simple format like:

```
_ 50
AH 120 50 100
_ 50
```

The underscores (`_`) are silence padding before and after.

---

## Step 5 — Run the Python Script to Generate All 34 WAV Files

Save this script as `~/mbrola_work/gen_wavs.py` and run it:

```python
#!/usr/bin/env python3
"""
gen_wavs.py — Generate 34 MBROLA us2 phoneme WAV files for TeensyTalkV2.

Requirements:
  - mbrola binary in PATH
    - us2 voice database in the same directory as this script

    Usage:
      cd ~/mbrola_work
        python3 gen_wavs.py

        Output:
          Creates a 'wavs/' subdirectory with 34 WAV files named per tts_phonemes.h
          """

          import os
          import subprocess
          import tempfile

          # Directory containing this script (where us2 lives)
          SCRIPT_DIR = os.path.dirname(os.path.abspath(__file__))
          US2_DB     = os.path.join(SCRIPT_DIR, "us2")
          OUT_DIR    = os.path.join(SCRIPT_DIR, "wavs")

          os.makedirs(OUT_DIR, exist_ok=True)

          # Each entry: (mbrola_phoneme, safe_wav_name, duration_ms)
          # Safe WAV names match exactly what tts_phonemes.h expects on the SD card.
          # Duration is approximate — long enough to hear the phoneme clearly.
          PHONEMES = [
              # Stops (unaspirated)
                  ("p",   "p.wav",    80),
                      ("b",   "b.wav",    80),
                          ("t",   "t.wav",    80),
                              ("d",   "d.wav",    80),
                                  ("k",   "k.wav",    80),
                                      ("g",   "g.wav",    80),
                                          # Stops (aspirated) — special FAT32-safe names
                                              ("p_h", "PH.wav",   90),
                                                  ("t_h", "TH2.wav",  90),
                                                      ("k_h", "KH.wav",   90),
                                                          # Flap
                                                              ("4",   "FL.wav",   60),
                                                                  # Fricatives
                                                                      ("f",   "f.wav",   100),
                                                                          ("v",   "v.wav",   100),
                                                                              ("s",   "s.wav",   100),
                                                                                  ("z",   "z.wav",   100),
                                                                                      ("S",   "S.wav",   100),   # "sh"
                                                                                          ("Z",   "Z.wav",   100),   # "zh"
                                                                                              ("T",   "T.wav",   100),   # "th" voiceless
                                                                                                  ("D",   "D.wav",   100),   # "th" voiced
                                                                                                      ("h",   "h.wav",    80),
                                                                                                          # Affricates
                                                                                                              ("tS",  "tS.wav",  110),   # "ch"
                                                                                                                  ("dZ",  "dZ.wav",  110),   # "j"
                                                                                                                      # Nasals
                                                                                                                          ("m",   "m.wav",   100),
                                                                                                                              ("n",   "n.wav",   100),
                                                                                                                                  ("N",   "N.wav",   100),   # "ng"
                                                                                                                                      # Liquids / glides
                                                                                                                                          ("l",   "l.wav",    90),
                                                                                                                                              ("r",   "r.wav",    90),
                                                                                                                                                  ("j",   "j.wav",    80),   # "y"
                                                                                                                                                      ("w",   "w.wav",    80),
                                                                                                                                                          # Syllabic r
                                                                                                                                                              ("r=",  "ER.wav",  130),
                                                                                                                                                                  # Monophthong vowels
                                                                                                                                                                      ("i",   "i.wav",   130),   # "ee"
                                                                                                                                                                          ("I",   "I.wav",   110),   # "ih"
                                                                                                                                                                              ("E",   "E.wav",   110),   # "eh"
                                                                                                                                                                                  ("{",   "AE.wav",  120),   # "ae" — FAT32-safe name
                                                                                                                                                                                      ("A",   "A.wav",   130),   # "ah"
                                                                                                                                                                                          ("V",   "V.wav",   110),   # "uh"
                                                                                                                                                                                              ("@",   "SCH.wav", 100),   # schwa — FAT32-safe name
                                                                                                                                                                                                  ("U",   "U.wav",   100),   # "oo" (short)
                                                                                                                                                                                                      ("u",   "u.wav",   130),   # "oo" (long)
                                                                                                                                                                                                          ("O",   "O.wav",   130),   # "aw"
                                                                                                                                                                                                              ("Or",  "OR.wav",  140),   # "or" — FAT32-safe name
                                                                                                                                                                                                                  # Diphthongs
                                                                                                                                                                                                                      ("EI",  "EI.wav",  150),   # "ay"
                                                                                                                                                                                                                          ("AI",  "AI.wav",  150),   # "eye"
                                                                                                                                                                                                                              ("OI",  "OI.wav",  150),   # "oy"
                                                                                                                                                                                                                                  ("@U",  "OW.wav",  150),   # "oh" — FAT32-safe name
                                                                                                                                                                                                                                      ("aU",  "aU.wav",  150),   # "ow"
                                                                                                                                                                                                                                      ]
                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                      def make_pho(phoneme, duration_ms):
                                                                                                                                                                                                                                          """Create a .pho file content for a single isolated phoneme."""
                                                                                                                                                                                                                                              return f"_ 50\n{phoneme} {duration_ms} 50 100\n_ 50\n"
                                                                                                                                                                                                                                              
                                                                                                                                                                                                                                              def generate_wav(phoneme, wav_name, duration_ms):
                                                                                                                                                                                                                                                  pho_content = make_pho(phoneme, duration_ms)
                                                                                                                                                                                                                                                      out_path = os.path.join(OUT_DIR, wav_name)
                                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                                          with tempfile.NamedTemporaryFile(mode='w', suffix='.pho', delete=False) as f:
                                                                                                                                                                                                                                                                  f.write(pho_content)
                                                                                                                                                                                                                                                                          pho_path = f.name
                                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                              try:
                                                                                                                                                                                                                                                                                      result = subprocess.run(
                                                                                                                                                                                                                                                                                                  ["mbrola", US2_DB, pho_path, out_path],
                                                                                                                                                                                                                                                                                                              capture_output=True, text=True
                                                                                                                                                                                                                                                                                                                      )
                                                                                                                                                                                                                                                                                                                              if result.returncode != 0:
                                                                                                                                                                                                                                                                                                                                          print(f"  ERROR generating {wav_name}: {result.stderr.strip()}")
                                                                                                                                                                                                                                                                                                                                                  else:
                                                                                                                                                                                                                                                                                                                                                              size = os.path.getsize(out_path)
                                                                                                                                                                                                                                                                                                                                                                          print(f"  OK  {wav_name:12s}  ({size:6d} bytes)  [{phoneme}]")
                                                                                                                                                                                                                                                                                                                                                                              finally:
                                                                                                                                                                                                                                                                                                                                                                                      os.unlink(pho_path)
                                                                                                                                                                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                                                                                                                                                                      print(f"Generating {len(PHONEMES)} phoneme WAV files into {OUT_DIR}/")
                                                                                                                                                                                                                                                                                                                                                                                      print(f"Using MBROLA database: {US2_DB}")
                                                                                                                                                                                                                                                                                                                                                                                      print()
                                                                                                                                                                                                                                                                                                                                                                                      
                                                                                                                                                                                                                                                                                                                                                                                      for (phoneme, wav_name, duration_ms) in PHONEMES:
                                                                                                                                                                                                                                                                                                                                                                                          generate_wav(phoneme, wav_name, duration_ms)
                                                                                                                                                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                                                                                                          print()
                                                                                                                                                                                                                                                                                                                                                                                          print("Done! Copy all files from wavs/ to the root of your SD card.")
                                                                                                                                                                                                                                                                                                                                                                                          ```
                                                                                                                                                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                                                                                                          Run it:
                                                                                                                                                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                                                                                                          ```bash
                                                                                                                                                                                                                                                                                                                                                                                          cd ~/mbrola_work
                                                                                                                                                                                                                                                                                                                                                                                          python3 gen_wavs.py
                                                                                                                                                                                                                                                                                                                                                                                          ```
                                                                                                                                                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                                                                                                          You should see output like:
                                                                                                                                                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                                                                                                          ```
                                                                                                                                                                                                                                                                                                                                                                                          Generating 45 phoneme WAV files into /Users/you/mbrola_work/wavs/
                                                                                                                                                                                                                                                                                                                                                                                          Using MBROLA database: /Users/you/mbrola_work/us2
                                                                                                                                                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                                                                                                            OK  p.wav         (  3244 bytes)  [p]
                                                                                                                                                                                                                                                                                                                                                                                              OK  b.wav         (  3244 bytes)  [b]
                                                                                                                                                                                                                                                                                                                                                                                                ...
                                                                                                                                                                                                                                                                                                                                                                                                  OK  aU.wav        (  5804 bytes)  [aU]
                                                                                                                                                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                                                                                                                                                  Done! Copy all files from wavs/ to the root of your SD card.
                                                                                                                                                                                                                                                                                                                                                                                                  ```

                                                                                                                                                                                                                                                                                                                                                                                                  ## Step 7 — FAT32 Filename Reference
                                                                                                                                                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                                                                                                                                                  Eight MBROLA phoneme names contain characters that are illegal on FAT32 filesystems.
                                                                                                                                                                                                                                                                                                                                                                                                  These are renamed as follows (this is already handled by `tts_phonemes.h`):
                                                                                                                                                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                                                                                                                                                  | MBROLA phoneme | WAV filename on SD card | Notes                  |
                                                                                                                                                                                                                                                                                                                                                                                                  |----------------|------------------------|------------------------|
                                                                                                                                                                                                                                                                                                                                                                                                  | `{`            | `AE.wav`               | "ae" vowel             |
                                                                                                                                                                                                                                                                                                                                                                                                  | `@`            | `SCH.wav`              | schwa                  |
                                                                                                                                                                                                                                                                                                                                                                                                  | `@U`           | `OW.wav`               | "oh" diphthong         |
                                                                                                                                                                                                                                                                                                                                                                                                  | `r=`           | `ER.wav`               | syllabic r             |
                                                                                                                                                                                                                                                                                                                                                                                                  | `p_h`          | `PH.wav`               | aspirated p            |
                                                                                                                                                                                                                                                                                                                                                                                                  | `t_h`          | `TH2.wav`              | aspirated t            |
                                                                                                                                                                                                                                                                                                                                                                                                  | `k_h`          | `KH.wav`               | aspirated k            |
                                                                                                                                                                                                                                                                                                                                                                                                  | `4`            | `FL.wav`               | alveolar flap          |
                                                                                                                                                                                                                                                                                                                                                                                                  | `Or`           | `OR.wav`               | "or" vowel             |
                                                                                                                                                                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                                                                                                                                                                  All other phoneme names (single ASCII letters or short letter pairs) are used as-is
                                                                                                                                                                                                                                                                                                                                                                                                  with `.wav` appended.

                                                                                                                                                                                                                                                                                                                                                                                                              ## Troubleshooting
                                                                                                                                                                                                                                                                                                                                                                                                              
                                                                                                                                                                                                                                                                                                                                                                                                              **`mbrola: command not found`**
                                                                                                                                                                                                                                                                                                                                                                                                              Run `which mbrola` to confirm installation. On Apple Silicon, you may need Rosetta:
                                                                                                                                                                                                                                                                                                                                                                                                              ```bash
                                                                                                                                                                                                                                                                                                                                                                                                              arch -x86_64 mbrola -h
                                                                                                                                                                                                                                                                                                                                                                                                              ```
                                                                                                                                                                                                                                                                                                                                                                                                              If that works, edit the `subprocess.run` call in `gen_wavs.py` to use
                                                                                                                                                                                                                                                                                                                                                                                                              `["arch", "-x86_64", "mbrola", ...]`.
                                                                                                                                                                                                                                                                                                                                                                                                              
                                                                                                                                                                                                                                                                                                                                                                                                              **`us2: no such file`**
                                                                                                                                                                                                                                                                                                                                                                                                              Make sure you downloaded the `us2` binary database and the script is run from the
                                                                                                                                                                                                                                                                                                                                                                                                              same directory (`~/mbrola_work/`).
                                                                                                                                                                                                                                                                                                                                                                                                              
                                                                                                                                                                                                                                                                                                                                                                                                              **Phoneme sounds wrong or robotic**
                                                                                                                                                                                                                                                                                                                                                                                                              The durations in `gen_wavs.py` are approximate starting points. You can tune
                                                                                                                                                                                                                                                                                                                                                                                                              individual durations in the `PHONEMES` list and re-run the script.
                                                                                                                                                                                                                                                                                                                                                                                                              
                                                                                                                                                                                                                                                                                                                                                                                                              **Teensy plays silence / no audio**
                                                                                                                                                                                                                                                                                                                                                                                                              - Confirm WAV files are in the SD card root (not a subdirectory).
                                                                                                                                                                                                                                                                                                                                                                                                              - Confirm filenames match exactly (case-sensitive on some cards).
                                                                                                                                                                                                                                                                                                                                                                                                              - Check Serial Monitor output — if phoneme sequence prints but no audio, check
                                                                                                                                                                                                                                                                                                                                                                                                                your I2S wiring and AudioPlayWav configuration in `TeensyTalkV2.ino`.
